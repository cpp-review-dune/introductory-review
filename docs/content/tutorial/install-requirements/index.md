## Node.js

- Download and install [Node.js](https://nodejs.org) (it includes npm) for your platform.

### Check Node.js install

{{< btn-copy text="node --version" >}}

```bash
node --version
```

### Check npm install

{{< btn-copy text="npm --version" >}}

```bash
npm --version
```
